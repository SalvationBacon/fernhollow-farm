# 🌾 Fernhollow Farm

A cozy pixel-art idle farming game that runs entirely in your browser. Plant crops, befriend villagers, explore the mine, fish the river, and rebuild the village — one harvest at a time.

## 🎮 Play Now

**[fernhollow-farm →](https://salvationbacon.github.io/fernhollow-farm/)**

Installable as an app on mobile — tap "Add to Home Screen" for the full experience.

## Features

- 🌱 **24 crops** across 4 seasons with quality tiers (silver, gold)
- 🎣 **Fishing minigame** — hold-and-release catch mechanic with 24 seasonal fish
- 🌿 **Foraging** — rock-paper-scissors vs the Forest Spirit for seasonal loot
- 🪓 **Woodcutting** — chop trees, collect wood and sap, upgrade your axe
- ⛏ **Mining** — 100 floors of ore, gems, and checkpoints
- 🐔 **Animals** — chickens, cows, sheep with daily products
- 🏛 **Town Hall** — donate to rebuild 4 rooms and unlock the forge, kitchen, and more
- 👥 **6 villagers** — gift, earn hearts, unlock shops and tools
- 🔨 **Forge** — smelt copper, iron, and gold ingots on timed queues
- ⭐ **45 achievements** across 9 categories
- 🎵 **Procedural folk music** — pentatonic pluck engine with day/night dynamics
- 📱 **PWA** — installable, works offline, autosaves to localStorage
- 🖥 **PC mode** — landscape sidebar layout for desktop

## Tech

Single HTML file. No frameworks, no build step, no server. Vanilla JS, CSS, and Web Audio API. Press Start 2P pixel font. ~7,300 lines.

## How It Works

Each real-world UTC hour = 1 in-game day. Crops grow in real time. Ship your harvest before the hour resets and collect your gold at midnight. Manage your 30 energy across farming, fishing, foraging, chopping, and mining.

## Install as App

- **Android**: Visit the link in Chrome → menu → "Install app" (or wait for the install banner)
- **iOS**: Visit in Safari → Share → "Add to Home Screen"
- **Desktop**: Chrome address bar → install icon
